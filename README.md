# Thezdin WoW Interface

## Requirements

* python3
* pip

---

## Setup

    pip install --user -r requirements.txt
    cp .env.example .env

## Config via  .env

Open `.env` and set values

**If you're using WSL, specify the `/mnt/...` path**

- WOW_PATH=/path/to/WoW
- SERVER={SERVER}-US
- CHARACTER=

## Run

    python bin/thezui
